微信读书神作榜单
=================

微信读书上的书籍浩如烟海、不知道怎么选书、那么我们可以用神作开始、就算你不喜欢，但是不会太差。

## 微信读书神作规则
	1.	评价大于500人
	2.	推荐值大于90%


## 主要功能


- 用Selenium从微信读书中获取所有的神作书籍
- 将数据写入本地sqlite数据库
- 由于微信读书神作榜单是每天更新、并且仅显示前一日的前200条书籍数据、所以我们需要每天定时跑一下程序


## 运行环境

- [Python 3](https://www.python.org/)

## 第三方库

- [Selenium](https://www.selenium.dev/)
